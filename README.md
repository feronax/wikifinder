# Wikifinder 🔍

**Wikifinder** est un jeu quotidien où tu dois deviner l'article Wikipedia du jour.

Le texte de l'article est affiché avec tous les mots masqués. Entre des mots un par un : chaque mot correct est révélé dans le texte. L'objectif est de trouver le titre de l'article en le moins de tentatives possible.

Une nouvelle partie est disponible chaque jour, en français et en anglais.

🎮 **Jouer** : [wikifinder.vercel.app](https://wikifinder.vercel.app)
